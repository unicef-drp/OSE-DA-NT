# Content Agreement Template

Use this template before producing a brief, slide deck, or polished narrative
output.

## Review Convention

Use a paired review file when commenting on or proposing edits to the agreement.

Suggested file pattern:

- main file: `name.md`
- review file: `name.review.md`

Comment syntax:

- any line starting with `>>>` at the top of the review file is a general
  comment for the whole document
- any line starting with `>>>` immediately under a copied line or paragraph
  applies to that line or paragraph above
- if you want to propose replacement text, just edit the copied text in the
  review file and the agent should compare the review file against the main file
  to detect the change

Example:

```md
>>> General: tighten this agreement before any PPT work starts.

## Questions To Answer

2. Which countries show the largest 10-year and 20-year reductions, and how do
	those changes compare with burden change and population growth pressure?
>>> Split this into prevalence change and burden change.

2. Which countries show the largest 10-year and 20-year reductions in modeled
	stunting prevalence?
3. Which countries show the largest reduction in the number of children
	affected, and where is progress offset by population growth?
```
